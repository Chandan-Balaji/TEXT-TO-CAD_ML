## ORIGINAL PYTHON TOOLS (Requires FreeCAD Setup)
=====================================================

NOTE: These require FreeCAD Python integration to be configured.
See FREECAD_SETUP.md for configuration steps.

### Generate Training Data (Create Dataset)

# Generate sphere training data
python training_data_generation/scripts/generate_sphere_data.py --num-datapoints 10000 --output datasets/sphere_data.csv

# Generate cube/box data
python training_data_generation/scripts/generate_box_and_cube_data.py --num-datapoints 10000 --output datasets/cube_data.csv

# Generate cylinder data
python training_data_generation/scripts/generate_cylinder_data.py --num-datapoints 10000 --output datasets/cylinder_data.csv

# Generate cone data
python training_data_generation/scripts/generate_cone_data.py --num-datapoints 10000 --output datasets/cone_data.csv

# Generate torus data
python training_data_generation/scripts/generate_torus_data.py --num-datapoints 10000 --output datasets/torus_data.csv

### Train the Model

# Create datasets directory first
mkdir datasets

# Train the model on generated data
python model_generation/scripts/train_model.py --datasets-dir datasets --epochs 50 --batch-size 64

# This will create:
# - model.pth (trained model)
# - config.yaml (configuration)
# - vectorizer.pkl (text vectorizer)

### Run Text-to-CAD Conversion

# Convert text description to 3D CAD file
python text_to_cad/scripts/run_text_to_cad.py \
    --text-input "A large sphere" \
    --output-file output_sphere.FCStd \
    --model-file model.pth \
    --config-file config.yaml \
    --vectorizer-file vectorizer.pkl

# Open the generated file in FreeCAD
# View -> Visibility -> Show all objects

### Generate Example 3D Shape (Standalone)

# Run the example shape generator
python generative_cad/scripts/example_generate_freecad_shape.py

# This creates: example_freecad_shape.fcstd
